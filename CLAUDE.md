## Project Overview - Band Practice App v3

Flask web app for managing band practice songs with lyrics, notes, and BPM.
Users create Songs Collections, linked to one or more Spotify playlists.
Auto-fetch lyrics from Genius, organize songs into Collections for different bands, add line-specific practice notes.
Deployed to Google Cloud Run with Firestore database.
